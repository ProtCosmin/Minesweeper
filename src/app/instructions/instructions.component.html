Rules of Minesweeper

Minesweeper is a single-player puzzle game where the objective is to uncover all the squares on a grid without detonating any mines.

Basic Rules:

  • The game is played on a grid with hidden mines.
  • Each square has a number that indicates how many mines are adjacent to that square.
  • You can click on squares to uncover them. If a square contains a mine, the game ends.
  • If a square is empty, it will show a number representing how many adjacent squares contain mines.
  • If you uncover all the squares without mines, you win the game!

Game Controls:

  • Left-click to uncover a square.
  • Right-click to flag a square that you think contains a mine.
